Document object model

DOM, Document Object Model, är ett programmeringsgränssnitt för webbläsare som representerar webbsidors struktur som en trädstruktur av noder.

DOM, eller Document Object Model, är ett programmeringsgränssnitt som tillhandahåller en strukturerad representation av dokumentet (vanligtvis en HTML- eller XML-fil) som en trädstruktur av noder. Varje element, attribut och text i dokumentet representeras som en nod i trädet. DOM möjliggör dynamisk interaktion med webbsidor genom att tillåta utvecklare att skapa, ändra och ta bort element och innehåll i realtid med hjälp av programmeringsspråk som JavaScript.

Genom att använda DOM kan utvecklare dynamiskt ändra strukturen, stilen och innehållet på en webbsida baserat på användarinteraktion eller andra händelser. Till exempel kan man lägga till nya element, ändra textinnehåll, ändra stilar och reagera på användarinteraktion som klick eller hovring över element.

DOM består av olika nivåer och gränssnitt som ger olika sätt att interagera med dokumentet. Det vanligaste är att använda DOM-nivå 3, som erbjuder omfattande funktioner för att navigera, manipulera och modifiera dokumentet på olika sätt. DOM är en central del av webbutveckling och är avgörande för att skapa interaktiva och dynamiska webbsidor.
